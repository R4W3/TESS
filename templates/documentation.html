1. Introduction


Welcome to Tess. This app aims to ease Home Automation and keep your data on your own server.
The following doc provides you with useful insights into the system and information on how to set it up.


2. Setup


2.1 Requirements


a. A Server (e.g. Raspberry Pi)

b. A functioning instance of MySQL on the Server


2.2 Start setup.py


Make sure your device for setup is in the same network as the server.
Open a shell and navigate to the downloaded directory. Enter the following command:

> python setup.py
Go to your browser of choice and access:
localhost:5000
Click the Go button to start the setup.
Enter your database details. Once you press next it will create a database called "tess". This will later contain usernames and passwords of your users. To view the entire database tree click here.

On the second screen you will create your admin account. This is the only role with full permissions. More on that topic later.
